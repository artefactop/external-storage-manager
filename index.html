External-storage-manager

Indicator for extract external drives at elementary OS

View project onGitHub

Description

This is a simple indicator build for elementary OS, this app allows you to extract or unmount your external drives easily.

Installation

You have to add my ppa and before install it.

sudo add-apt-repository ppa:jlnavarro111/jlnavarro111
sudo apt-get update && sudo apt-get install external-storage-manager
